How to run:
Run it in this order

1. change ip address
   run ipconfig in your cmd prompt
   look for IPv4 address
   go into frontend/lib/config.js
   paste ip here http://PASTE_IP_HERE:5000
   save file

2. backend
   open terminal
   cd into backend -> cd ./backend
   run -> node server.js

3. frontend
   open new terminal
   cd into frontend -> cd ./frontend
   run npx expo start
